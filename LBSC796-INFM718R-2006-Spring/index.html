Catalog Description

Principles of organizing and providing access to information using automated information storage and retrieval systems. Retrieval system models, index language selection, data structures, user interfaces and evaluation for text and multimedia applications.

Course Goals

  • Appreciate the capabilities and limitations of information retrieval systems.
  • Understand the design and implementation of retrieval systems for text and other media.
  • Evaluate the performance of an information retrieval system.
  • Situate information retrieval systems within a broader information seeking environment.
  • Identify current research problems in information retrieval.

Visual affordance/transfer effect: dictionary-inspired design indicates what user is looking at and what type of information this site will provide.

User interface: red links stand out from the subtle, non-obtrusive design, enabling repeat users to access definitions quickly. Contextual information helps first-time users understand the site's functionality.

Different images for each topic serve as visual markers and navigation aids.

Simple, subtle design uses consistency, navigational cues, alignment, and repetition.

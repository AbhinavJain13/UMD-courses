MORE ABOUT BOOKTALKS

Welcome to Children's Booktalks. This site has been designed as a resource for creating and using booktalks. These powerful tools are a great way to "sell" books to library patrons and potential readers. Throughout this site you will find video slideshows accompanying sample booktalks, written text for additional talk, tips on creating and using booktalks, and resources for gaining more ideas.

Video slideshows are in Quicktime format and were created using Imovie. Voices and individuals pictured are volunteers connected with the project, using freely available booktalk text.

This site was designed as a class project for Dr. Lin's LBSC 690 Information Technology class at the University of Maryland. It is intended to be a stand-alone site, but was created as a potential addition to other children's library web sites. Content was intended in-class presentation only.
